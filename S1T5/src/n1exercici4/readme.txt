Para compilar y ejecutar este ejercicio, debes seguir estos pasos:

1. Ir a la carpeta src, la carpeta anterior al package n1exercici4.

2. Copiar la ruta de la carpeta.

3. Abrir la CMD y escribir el comando
	cd larutaquehemoscopiado
	
Si hay espacios en la ruta, poner toda la ruta entre comillas.

Ahora ya estamos dentro de la carpeta

4. Compilamos los dos archivos, a traves del packcage, en este orden:
	javac n1exercici4\Programa.java
	javac n1exercici4\App.java


5. Ejecutamos el programa con el comando 
	java n1exercici4.App
